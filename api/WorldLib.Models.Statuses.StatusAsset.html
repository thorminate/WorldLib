RenderCheck

Determines whether this status effect should be visually rendered for a given actor.

public DelegateBridge<RenderEffectCheck, RenderEffectCheck> RenderCheck

Field Value

DelegateBridge<RenderEffectCheck, RenderEffectCheck>

Remarks

This delegate is evaluated before any visual effect instance is created. If multiple delegates are registered, all will be invoked and the last return value in the multicast invocation chain will be used by the game. Returning false prevents the effect from being visually rendered, but does not remove or disable the gameplay effect itself.
