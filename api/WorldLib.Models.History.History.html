Provides the last 2000 world logs in the form of a IReadOnlyList<T>.

public IReadOnlyList<HistoryEntry> Messages()

Returns

IReadOnlyList<HistoryEntry>

A IReadOnlyList<T> containing the last 2000 world logs.

Remarks

Fetching world logs involves querying the SQL database, so this operation can be Slow!

Only the last 2000 entries are retrieved at a time.
